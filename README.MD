# War-Game
A videogame adaptation of a strategy war game I made.

It is still in progress.

You control an army in a WWI-like war. You must conquer as many cities to build reinforcements and advance in technology while enemies attack you from every side.

## Current Features
* Hexgon Tile Game Board that can be customized to any size
* All units currently listed in the Wiki
* Map Rendering

## Planned Features
* Unit Rendering
* Unit Special Effects
* Main Gameplay Loop
* AI
* Different Types of Terrain
* A Technology advancement system
